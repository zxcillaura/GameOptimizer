# GameOptimizer — Игровой помощник

Мощная утилита для оптимизации Windows под максимальную производительность в играх.

## Возможности

- **Панель диагностики** - полная информация о системе
- **20+ системных твиков** - повышение FPS и снижение задержки
- **DNS Jumper** - смена DNS серверов (Cloudflare, Google, etc.)
- **Очистка системы** - удаление мусора и кэша
- **Управление античитами** - настройка FACEIT AC и Vanguard
- **Оптимизация игр** - готовые профили для CS2, Valorant, Apex
- **Система профилей** - быстрое применение наборов твиков
- **Предпросмотр изменений** - смотрите что будет изменено перед применением
- **Современный интерфейс** - минималистичный Fluent Design

## Установка и запуск

1. Скачай последний релиз
2. Запусти `GameOptimizer.exe` **от имени администратора**

## Сборка из исходников

```bash
dotnet restore
dotnet build -c Release
dotnet publish -c Release -o ./publish
```

## Новые возможности в версии 2.0

### Модернизированный интерфейс
- **Fluent Design** - современный минималистичный стиль
- **Тёмная и светлая темы** - полная поддержка обеих режимов
- **Система цветов** - улучшенная иерархия и контраст
- **Закруглённые углы** - элегантные компоненты (RoundedPanel, RoundedButton)

### Система профилей
- **Конкурентные игры** (CS:GO, Valorant, Apex) - максимум FPS + минимум пинга
- **Одиночные игры** (AAA тайтлы) - баланс графика/производительность
- **Стриминг** (Twitch/YouTube) - оптимизация CPU для трансляции
- **Очистка системы** - удаление ненужных сервисов и файлов
- **Кастомный профиль** - ручной выбор твиков

### Система предпросмотра
- Показывает все изменения перед применением
- Отмечает опасные операции красным
- Разделяет безопасные и рискованные действия
- Предоставляет детальную сводку

### UI Helpers
- Унифицированный стиль всех элементов
- Легкая адаптация к темам
- Стандартные функции для сообщений и статусов
