# 🚀 GameOptimizer — Игровой помощник

Мощная утилита для оптимизации Windows под максимальную производительность в играх.

## ✨ Возможности

- 📊 Панель диагностики системы
- ⚙️ 20+ системных твиков для повышения FPS
- 🌐 DNS Jumper с 30+ серверами
- 🧹 Очистка системы от мусора
- 🛡️ Управление античитами (FACEIT, Vanguard)
- 🎮 Оптимизация игр (CS2, Valorant)

## 🛠️ Установка и запуск

1. Скачай последний релиз
2. Запусти `GameOptimizer.exe` **от имени администратора**
