# zxcillaura Game Booster Pro

Утилита для оптимизации Windows 10/11 под игры, настройки сети и управления античитами (FACEIT, Riot Vanguard). Требует прав администратора.

## 🚀 Возможности
*   **Оптимизация:** Быстрая настройка питания, отключение анимаций, Game Bar, телеметрии и фоновых служб.
*   **Профили:** "Жёсткая" (максимум FPS) и "Мягкая" (безопасная) оптимизация.
*   **Сеть:** Встроенный DNS Jumper с тестированием пинга (Cloudflare, Google, AdGuard и др.) и отключение энергосбережения сетевых карт.
*   **Античиты:** Управление Hyper-V, VBS, DEP, службами FACEIT и Vanguard.
*   **Очистка:** Удаление временных файлов, кэша, Prefetch, Shader Cache и мусорных приложений.

## 🔧 Установка и запуск
1. Скачайте файл `GameOptimizer.exe` из раздела **Releases**.
2. Запустите файл **от имени Администратора** (нажмите на него правой кнопкой мыши → "Запуск от имени администратора").
3. Выберите нужный профиль или настройте параметры вручную.

## ⚠️ Внимание
Программа вносит изменения в реестр Windows и системные службы. Перед использованием настоятельно рекомендуется создать точку восстановления системы (это можно сделать прямо в программе, на вкладке "Главная").
